# height-balanced binary search tree


(数据结构)



定义:
一个高度平衡树，也是一个二叉搜索树。它支持成员关系、插入和删除操作，时间以树中节点数的对数为单位。



概括(我是一种……)
高度平衡树，二叉搜索树，平衡二叉树。



专业化(…是一种我。)
AVL树，红黑树。
