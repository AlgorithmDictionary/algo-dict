# hash function

(算法)



定义:
将键映射为整数的函数，通常是为了在较小的值集上得到均匀分布。



也称为散列。



专业化(…是一种我。)
不同种类:线性哈希，完美哈希，最小完美哈希，保序最小完美哈希，具体功能:皮尔逊哈希，乘法法。



聚合父级(我是…的一部分或在…中使用)
哈希表，统一哈希，通用哈希，布隆过滤器，默克尔树，HyperLogLog，位置敏感哈希。



另请参阅
简单的统一哈希。



注意:
整数的范围通常为[0…m-1]，其中m是质数或2的幂。
