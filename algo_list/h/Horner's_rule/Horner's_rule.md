# Horner's rule


(算法)



定义:
多项式A(x) = a0 + a1x + a2x²+ a3x³+…可以写成A(x) = a0 + x(a1 + x(a2 + x(a3 +…)))。



注意:
一个多项式可以在点x'处求值，即A(x')在Θ(n)时间内使用Horner规则计算。也就是说，重复的乘法和加法，而不是简单的x的幂，乘以系数，累积的方法。
