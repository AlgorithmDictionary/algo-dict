# Hamiltonian cycle


(定义)



定义:
一种通过图的路径，它从同一顶点开始和结束，并且只包含其他顶点一次。



也被称为旅游。



概括(我是一种……)
周期。



专业化(…是一种我。)
旅行推销员。



另请参阅
哈密顿路径，欧拉循环，车辆路线问题，完美匹配。



注意:
一个哈密顿循环包含每个顶点一次;欧拉循环包含每条边一次。

也被称为哈密顿电路。

以威廉·罗文·汉密尔顿爵士(1805-1865)命名。
