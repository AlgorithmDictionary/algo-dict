# Euclidean traveling salesman problem


(经典问题)



定义:
求一个平面上各点之间的最小欧氏距离的路径，使每个点都经过一次并返回到它的起始点。



另请参阅
旅行推销员，生成树。



注意:
这可以推广到更高的维度，例如，三维空间中的点。由于点与点之间的代价是平面距离，而不是任意的权值，所以这个问题是旅行推销员问题的特殊情况。
