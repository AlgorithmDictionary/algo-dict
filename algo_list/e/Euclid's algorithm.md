# Euclid's algorithm


(算法)



定义:
一种计算两个正整数的最大公约数的算法。它是Euclid(a,b){if (b=0) then return a;否则返回Euclid(b, a mod b);}。运行时复杂度为O((log a)(log b))位操作。



又称欧几里得算法。



另请参阅
二进制GCD，扩展欧几里得算法，Ferguson-Forcade算法。
