# external index


(数据结构)



定义:
一种添加到主数据结构中的辅助数据结构，以改进操作，如在辅助键上进行搜索。



另请参阅
倒排索引，哈希表，搜索树。



注意:
实现可能会在主数据结构中嵌入外部索引的部分，例如向每个节点添加一些字段。如果向主数据结构添加或删除项，也必须维护外部索引。数据可能有多个外部索引，比如一个电影数据库，其标题、演员和类型的索引是反向的。
