# kth shortest path


(经典问题)



定义:
寻找图中从一个顶点到另一个顶点的第k条最短路径的问题。变量可能要求路径是边或顶点不相交的，即不共享任何边或顶点。“最短”可能是边数最少，总权重最少等。



专业化(…是一种我。)
最短路径。
