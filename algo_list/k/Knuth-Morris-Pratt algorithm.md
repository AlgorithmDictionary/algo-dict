# Knuth-Morris-Pratt algorithm


(算法)



定义:
字符串匹配算法将搜索字符串转换为有限状态机，然后使用要搜索的字符串作为输入字符串运行该机器。执行时间为O(m+n)，其中m为搜索字符串的长度，n为待搜索字符串的长度。



也被称为KMP。
