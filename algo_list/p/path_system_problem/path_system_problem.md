# 路径系统问题


(经典问题)



定义:
对于路径系统P=(x,R,S,T)，其中S对x、T对x、R对x × x × x, S中是否存在可容许的顶点，当且仅当x∈T，或存在可容许的y,z∈x，使得(x,y,z)∈R。
