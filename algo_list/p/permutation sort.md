# permutation sort


(算法)



定义:
一种效率极低的排序算法，它生成每一种元素的排列，直到所有元素都按顺序排列。



概括(我是一种……)
排序算法。



另请参阅
Bogosort, bozo sort, taco sort。



注意:
预期运行时间为Θ(n!)。
