# Post的通信问题


(经典问题)



定义:
给定一组字符串对，找到一个字符串对序列，使得所有第一个字符串对的连接与所有第二个字符串对的连接是相同的。这是一个无法确定的问题。



也被称为PCP。


作者:SKS


Emil Post，递归不可解问题的变体。美国数学学会通报，52(2):264-268,1946。
