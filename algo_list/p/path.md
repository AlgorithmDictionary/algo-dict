# 路


(定义)



定义:
图的顶点列表，其中每个顶点与下一个顶点之间有一条边。



专业化(…是一种我。)
简单路径，最短路径，循环，哈密顿循环，欧拉循环，交替路径。



另请参阅
所有对最短路径，所有简单路径。



注意:
除非另有定义，否则通常假定路径为简单路径。

在有向图中，每个顶点都有一条从它到下一个顶点的边。
