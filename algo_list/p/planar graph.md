# 平面图


(定义)



定义:
可以在平面上画出没有交叉边的图形。



概括(我是一种……)
图。



专业化(…是一种我。)
对偶，平面直线图。



注意:
等价地，一个图不包含任何子图同胚于5个顶点上的完全图或每个分区上有3个顶点的完全二部图。
