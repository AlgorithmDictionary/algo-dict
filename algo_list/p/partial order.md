# 偏序


(定义)



定义:
为某些(但不一定是全部)物品对定义的顺序。例如，集合{a, b}和{a, c, d}是{a, b, c, d}的子集，但两者都不是对方的子集。所以"的子集"是集合上的偏序。



形式定义:偏序是一种自反的、传递的、反对称的二元关系。
