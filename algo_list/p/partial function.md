# 偏函数


(定义)



定义:
对于某些正确类型的输入，也就是说，对于某个域的某些输入，没有定义的函数。例如，除法是一个偏函数，因为除以0是未定义的(在实数上)。



另请参阅
全函数，部分递归函数。



注意:
这个定义说一个函数要么是部分的，要么是全部的，但不是两者都是。一些作者认为所有的函数都是偏函数，但也有一些是全函数。
