# 原始算法


(定义)



定义:
一种算法，其中所有可计算的步骤都是基本操作。



概括(我是一种……)
算法。



另请参阅
复合算法，原始递归。



注意:
任何算法要么是原始算法，要么是复合算法。原语和复合语之间的区别多少有些主观。
