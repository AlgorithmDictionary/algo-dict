# 指针机


(定义)



定义:
一种计算模型，其内存由由指针连接的寄存器或记录的无界集合组成。每个寄存器可以包含任意数量的附加信息。不允许使用任何算术来计算寄存器的地址。访问寄存器的唯一方法是遵循指针。
