# Johnson's algorithm


(算法)



定义:
一种求解稀疏加权有向图中所有对最短路径问题的算法。首先，它添加一个从它到所有其他节点的零权边的新节点，并运行Bellman-Ford算法来检查负权循环并找到h(v)，从新节点到节点v的路径的最小权值。然后它使用节点的h(v)值重新加权边。最后，对于每个节点，它运行Dijkstra算法并将计算的最小权重存储到其他节点，使用节点的h(v)值重新加权作为最终权重。时间复杂度为O(V²log V + VE)。
