# 转置顺序搜索


(算法)



定义:
通过每次检查一个项来搜索数组或列表。如果找到了该值，将其与之前的值交换，以便下次更快地找到它。



也称为自组织顺序搜索。



概括(我是一种……)
线性搜索，自组织启发式[维基百科]。



另请参阅
移动到前面启发式，自组织列表。



注意:
这将把更频繁搜索的项目移到前面。如果有一些东西比其他东西更常被寻找，这可以节省时间。但是，二分查找或哈希表几乎总是更快。
