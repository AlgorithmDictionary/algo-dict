# 螺纹的树


(数据结构)



定义:
一种二叉搜索树，其中每个节点使用一个其他情况下为空的左子链接来引用该节点的有序前代节点，并使用一个空的右子链接来引用其有序后继节点。



另请参阅
右线程树，二叉搜索树。



注意:
参见[knnuth97, 1:20 20, Sect. 2.3.1]。虽然该定义可以引用多路树，但Knuth将线程树定义为二叉树。
