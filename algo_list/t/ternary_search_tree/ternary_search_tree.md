# 三元搜索树


(数据结构)



定义:
一个3向树，其中每个节点的左子树的键值小于节点的键值，每个中间子树的键值等于节点的键值，每个右子树的键值大于节点的键值。如果键是多键(字符串、数组、列表等)，则中间子树按下一个子键(字符、数组或列表项等)组织。



也被称为TST。



概括(我是一种……)
k=3的k元树，搜索树。



另请参阅
trie，二叉搜索树，多键快速排序。
