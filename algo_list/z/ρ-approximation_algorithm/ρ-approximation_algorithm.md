# rho近似算法


(算法技术)



定义:
一种近似算法，保证在最大(或至少，视情况而定)ρ乘以最优值时找到一个解。比值ρ是算法的性能比或相对性能保证。



概括(我是一种……)
近似算法。



另请参阅
蒙特卡罗算法。
