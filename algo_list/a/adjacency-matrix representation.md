# 邻接矩阵表示


（数据结构）



释义
使用n×n矩阵表示具有n个顶点的有向图，其中（i，j）处的项为1，如果存在从顶点i到顶点j的边；否则该条目为0。可以使用权重作为条目来表示加权图。无向图可以使用（i，j）和（j，i）中的相同条目或使用上三角矩阵来表示。
