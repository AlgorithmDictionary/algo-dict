# AVL树


（数据结构）



释义
一种平衡的二进制搜索树，其中一个节点的两个子树（子树）的高度最多相差一个。查找、插入和删除是O（logn），其中n是树中的节点数。



泛化（我是一种…）
高度平衡树、平衡二叉树、二叉搜索树、红黑树（着色时）。



聚合子项（…是我的一部分或用于我。）
左旋转、右旋转。



另请参阅
B树，螺纹树，斐波那契树。



笔记
该结构以发明人Adelson Velskii和Landis的名字命名。如有必要，在插入或删除后使用旋转重新平衡树。

加里·格鲁布之后<ggrubb@sr.hp.com>.

AVL树至少和红黑树一样平衡。
