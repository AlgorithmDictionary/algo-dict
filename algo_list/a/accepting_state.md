# 接受状态


释义



释义
如果有限状态机完成输入字符串并处于接受状态，则该字符串被接受或被认为是有效的。



另请参阅
识别器。



笔记
机器可以进入接受状态，然后离开。接受取决于机器是否在接受状态下完成。
