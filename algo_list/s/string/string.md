# 字符串


(数据结构)



定义:
字符列表，通常作为数组实现。非正式用语:单词、短语、句子等由于文本处理非常常见，所以通常可以使用带有子字符串操作的特殊类型。



另请参阅
字符串匹配，有错误的字符串匹配，最长公共子序列，字符串结束，文本。



注意:
术语字符串通常指的是一小串字符，比如一个名字或一个句子。文本一词通常指一长串字符，如一篇文章或一本书。
