# 对称集差分


(定义)



定义:
在任意一个集合中，但不在两个集合中的成员。也就是说，对于集合A和B，它是(A - B)∪(B - A)。



另请参阅
的区别。



注意:
{2,3,5} Δ{7,5} ={2,3,7}。
