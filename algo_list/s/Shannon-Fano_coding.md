# 香农-范诺编码


(算法)



定义:
基于每个字符出现频率的可变长度编码。将字符分成两个集，每个集的频率尽可能接近一半，并为这些集分配0或1编码。重复划分集合，直到每个字符都有一个唯一的编码。



另请参阅
算术编码，霍夫曼编码，齐夫定律。



注意:
香农-范诺是一个最小前缀码。霍夫曼最适合字符编码(一个字符一个码字)和简单的程序。算术编码更好，因为它可以分配小数位，但更复杂，并且有专利。
