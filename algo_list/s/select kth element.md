# 选择第k个元素


(经典问题)



定义:
求集合的第k个最小元素。两种方法是修改分布排序或选择和分区。



也被称为选择问题，找到第k个最小元素，第k个最小元素。



另请参阅
选择，修改查找，查找，储层采样。



注意:
选择和分区大约将每个步骤中必须考虑的元素数量分成两半。修改后的分布排序将每一步的元素数除以m。后者更快，但前者更普遍适用。
