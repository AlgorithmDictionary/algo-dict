# 斯坦纳树


(定义)



定义:
在一个无向的加权图或空间中的点中，连接一组指定的顶点(称为终端)的最小权树。树可以包含非终结点，称为斯坦纳顶点或斯坦纳点。



专业化(…是一种我。)
欧氏斯坦纳树，直线斯坦纳树。



另请参阅
最小生成树。



注意:
它与最小生成树的不同之处在于斯坦纳顶点的集合必须被识别。也就是说，可以使用额外的顶点。

以雅各布·施泰纳命名。

一些作者区分了斯坦纳树和最小斯坦纳树。
