# 施泰纳比


(定义)



定义:
对于给定的斯坦纳树问题的变体，一组终端的最小生成树的长度与同一组终端的最优斯坦纳树的长度的最大可能之比。通常写成ρ ()



注意:
在图中ρ = 2。在欧几里得度规中，ρ = 2/√(3)在直线度规中，ρ = 3/2。
