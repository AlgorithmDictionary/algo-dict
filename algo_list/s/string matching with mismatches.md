# 字符串匹配不匹配


(算法)



定义:
字符串匹配错误的特殊情况，其中不匹配是唯一允许的错误类型。



概括(我是一种……)
字符串匹配错误。



专业化(…是一种我。)
不匹配的暴力字符串搜索。
