# 搜索树


(数据结构)



定义:
一种树，其中一个节点的每一个子树的键值小于其右侧节点的任何其他子树。节点中的键在概念上位于子树之间，大于其左侧子树中的任何键，小于其右侧子树中的任何键。



专业化(…是一种我。)
二叉搜索树，b树，(a, b)-树，三叉搜索树。



另请参阅
搜索树性质，移动到根启发式，k-d树。



注意:
同时也是二叉树的搜索树就是二叉搜索树。
