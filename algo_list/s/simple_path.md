# 简单路径


(定义)



定义:
不重复任何顶点的路径，除了第一个顶点和最后一个顶点可能是相同的顶点。



专业化(…是一种我。)
哈密顿路径，循环。



另请参阅
枚举两个顶点之间的所有简单路径。



注意:
2017年8月，定义改变为允许第一个和最后一个顶点相同，与Knuth一致[Knuth98, 1:363]。在此之前，定义是“不重复任何顶点的路径”。
