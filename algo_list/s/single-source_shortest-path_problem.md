# 单源最短路径问题


(经典问题)



定义:
在一个加权的有向图中，找到从一个特定的源顶点到每一个其他顶点的最短路径。Dijkstra算法在所有权重都是非负的情况下解决了这个问题。Bellman-Ford算法可以处理任意权重。
