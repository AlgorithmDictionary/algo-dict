# 字符串匹配错误


(算法)



定义:
在文本字符串中搜索模式字符串的近似出现(例如，最多预定义数量的符号不匹配、插入和删除)。预处理，例如建立索引，可能是允许的，也可能是不允许的。



也称为近似字符串匹配。



专业化(…是一种我。)
Ratcliff/Obershelp模式识别，语音编码，Jaro-Winkler, Levenshtein距离，不匹配的字符串匹配。
