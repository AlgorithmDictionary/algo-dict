# 异或


(定义)



定义:
“异或”或“不等于”功能:0 XOR 0 = 0,0 XOR 1 = 1,1 XOR 0 = 1,1 XOR 1 = 0。



也称为exclusive or。



另请参阅
And, or, not，暗示。



注意:
名称“排他或”与“包括或”相反。
