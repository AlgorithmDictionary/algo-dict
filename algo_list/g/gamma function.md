# gamma function


(定义)



定义:
n的函数，写成Γ(n)，是∫0∞e-xxn-1dx。递归Γ(n+1) = nΓ(n)。对于非负整数Γ(n+1) = n!



另请参阅
斯特林公式。



注意:
函数为所有数字定义，而阶乘(严格地)仅为非负整数定义。
