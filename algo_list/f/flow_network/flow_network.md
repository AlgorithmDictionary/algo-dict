# flow network


(定义)



定义:
一个加权的有向图，有两个特殊标记的节点，源s和汇t，以及一个将边映射到正实数的容量函数，u: E |→1 +。



另请参阅
有向图，最大流量问题，流量函数，阻塞流。



注意:
通过添加一个对所有源具有无限容量的超源或一个对所有汇具有无限容量的超链路，可以将具有多个源或多个汇的图转换为流网络。
