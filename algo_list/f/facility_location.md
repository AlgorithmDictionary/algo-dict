# facility location


(经典问题)



定义:
给定一组需求点、一个距离函数和一个参数p，找到一组p个供应对象(点、线、段等)，使某个距离目标函数最小化。该函数可以是任何需求点与最近供应点之间的最大距离，因此没有需求点离供应点太远，或者是到最近供应点的距离之和。

可容设施位置。
