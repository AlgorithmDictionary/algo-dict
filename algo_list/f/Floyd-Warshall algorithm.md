# Floyd-Warshall algorithm


(算法)



定义:
一种解决加权有向图中所有对最短路径问题的算法，方法是将图的邻接矩阵表示乘以多次。边可以有负权值，但不能有负权环。时间复杂度为Θ (V³)。
