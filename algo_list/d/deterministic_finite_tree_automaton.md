# deterministic finite tree automaton


(定义)



定义:
一种确定性有限状态机，它接受有限树而不仅仅是字符串。树节点用自动机的字母表中的字母标记，过渡函数编码树的每个分支的下一个状态。验收条件作相应修改。



也被称为DFTA。
