# dynamic array


(数据结构)



定义:
其大小可能随时间变化的数组。不仅添加或删除项目，而且使用的内存也会发生变化。



注意:
例如，Visual Basic中的REDIM或c中的malloc()。在某些语言(如Perl)中，所有数组都是动态的。
