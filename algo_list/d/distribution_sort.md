# distribution sort


(算法)



定义:
将项从输入分配到多个中间结构的任何排序算法，然后将这些中间结构收集并放置在输出中。



概括(我是一种……)
排序。



专业化(…是一种我。)
桶排序，线性探测排序，洗牌排序，归并排序，基数排序，反洗牌排序，受限全域排序，链排序，分配分区排序。



注意:
许多分布排序也是归并排序，这取决于如何执行分布。
