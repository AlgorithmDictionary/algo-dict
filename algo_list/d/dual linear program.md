# dual linear program


(定义)



定义:
每个线性规划都有一个对应的线性规划，称为对偶。max {b·y | ATy≤c且y≥0}。对于原线性规划的任意解x和对偶的任意解y，我们有c·x≥(AT y)T x = yT(Ax)≥y·b。对于最优x和y，等式成立。对于一个用整数线性规划表述的问题，该规划松弛的对偶的解可以作为见证。
