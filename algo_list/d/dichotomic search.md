# dichotomic search


(定义)



定义:
在每个步骤中通过在两个不同的选择(二分类)之间进行选择来搜索。



概括(我是一种……)
搜索。



专业化(…是一种我。)
二进制搜索。



注意:
分成多个备选方案是polychotomy。
