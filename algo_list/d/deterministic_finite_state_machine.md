# deterministic finite state machine


(定义)



定义:
一种有限状态机，每个符号和状态最多有一个转换。



也被称为DFA，确定性有限自动机。



另请参阅
不确定有限状态机。



注意:
有些领域，比如模型检查，要求(并假设)每个符号和状态只有一个转换。对于每个符号和状态至少有一个转换的机器有时被称为“完整的”。
