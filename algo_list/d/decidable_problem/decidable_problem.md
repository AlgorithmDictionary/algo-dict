# decidable problem


(定义)



定义:
一种决策问题，可以通过在有限的步骤中停止所有输入的算法来解决。与之相关的语言称为可判定语言。



也被称为完全可决问题，算法可解，递归可解。



另请参阅
不确定问题，NP, NP完全，可解，可处理，可计算。



注意:
为了强调，有时使用完全可决定问题的等价术语。
