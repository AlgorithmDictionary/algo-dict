# deterministic tree automaton


(定义)



定义:
一种确定性有限状态机，它接受无限的树而不仅仅是字符串。树节点用自动机的字母表中的字母标记，过渡函数编码树的每个分支的下一个状态。这种自动机的表达能力取决于树的接受条件。



另请参阅
确定性有限树自动机。



注意:
它们被用于分支时间时间逻辑、二阶多后继逻辑等的各种研究中。
