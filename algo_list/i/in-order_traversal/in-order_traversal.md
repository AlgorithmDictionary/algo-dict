# in-order traversal


(算法)



定义:
通过递归处理左子树，然后处理根，最后处理右子树来处理树的所有节点。



也称为对称遍历。



概括(我是一种……)
树遍历，深度优先搜索。



聚合父级(我是…的一部分或在…中使用)
treesort(1)。



另请参阅
后序遍历，预序遍历，级序遍历，丘比特-贾尼尼树遍历。
