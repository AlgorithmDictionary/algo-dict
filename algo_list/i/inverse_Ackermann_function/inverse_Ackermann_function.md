# inverse Ackermann function


(算法)



定义:
一个有两个参数的函数，它的值增长得非常非常慢。



形式定义:α(m,n) = min{i≥1:A(i，⌊m/n⌋)> log2 n}，其中A(i,j)为Ackermann函数。



也被称为α。



另请参阅
阿克曼函数。



注意:
这并不是阿克曼函数的严格逆函数。相反，它的增长速度和阿克曼函数的增长速度一样慢。
