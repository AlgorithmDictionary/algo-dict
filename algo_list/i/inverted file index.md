# inverted file index


(数据结构)



定义:
一种倒排索引，它只指示出现单词的文本，而不指示单词在文本中出现的位置。



概括(我是一种……)
反向索引。



另请参阅
全倒排索引，块寻址索引。



注意:
请参阅倒排索引中的示例。
