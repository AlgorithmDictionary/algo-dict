# introspective sort


(算法)



定义:
一种快速排序的变体，对于病态输入切换到堆排序，也就是说，当执行时间变成二次的时候。



也称为内向排序。


作者:钢


刘建平，基于自适应的排序选择算法，软件实践与经验，第8期，1997。
