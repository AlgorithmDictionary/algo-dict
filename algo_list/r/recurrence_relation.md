# 递归关系


(定义)



定义:
用序列中较早的值和基值来指定一个值序列。



也被称为递归方程。



另请参阅
掌握定理。



注意:
斐波那契数可以用递归关系F(n) = F(n-1) + F(n-2)来描述，其中F(1)=1, F(2)=1。执行时间通常通过建立然后求解一元递归关系来计算，例如T(n) = 2T(n/4) + 2。
