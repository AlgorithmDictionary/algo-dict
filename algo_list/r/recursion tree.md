# 递归树


(定义)



定义:
通过绘制递归函数调用的图表来分析算法复杂性的一种方法。



正式定义:阶为p的递归树T(p)要么为(i)空，要么为(ii)有p个子递归树。



注意:
也被称为“r树”。
