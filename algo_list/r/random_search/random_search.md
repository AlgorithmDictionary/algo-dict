# 随机搜索


(算法)



定义:
随机重复检查数组中的项，直到成功为止。



概括(我是一种……)
搜索，拉斯维加斯算法。



另请参阅
跳跃表，线性搜索，二分搜索。



注意:
如果需要某个具有特定值的项，并且有许多具有该值的项，则随机搜索比线性搜索的期望时间更短。如果并行处理是可用的，那么随机搜索是容错的(处理器可以停止工作而不会导致故障)并且相当有效。
