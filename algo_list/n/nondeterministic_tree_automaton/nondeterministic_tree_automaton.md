# 不确定性树自动机


(定义)



定义:
一种不确定的有限状态机，它接受无限的树而不仅仅是字符串。树节点用自动机的字母表中的字母标记，过渡函数编码树的每个分支的下一个状态。这种自动机的表达能力取决于树的接受条件。
