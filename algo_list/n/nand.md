# nand


(定义)



定义:
否定连接:0 NAND 0 = 1,0 NAND 1 = 1,1 NAND 0 = 1,1 NAND 1 = 0。



另请参阅
并且，不是，or, xor。



注意:
NAND相当于AND后跟NOT。这是特别有趣的，因为所有可能的布尔函数都可以用NAND的组合来构建，并且NAND门很容易构建。
