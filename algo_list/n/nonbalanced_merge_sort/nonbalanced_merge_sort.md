# 非平衡归并排序


(算法)



定义:
一种k-way归并排序，其中输入和输出数据流的数量对于任何特定的遍历都是不同的。通常P个输入流被合并并分布到T个输出流，然后是T个输入流的合并和P个输出流的分布。



另请参阅
归并排序，简单归并排序，平衡归并排序。
