# 也不


(定义)



定义:
负分离:0 NOR 0 = 1, 0 NOR 1 = 0, 1 NOR 0 = 0, 1 NOR 0 = 0。



注意:
NOR相当于OR后面跟着NOT。这很有趣，因为所有可能的布尔函数都可以用NORs的组合来构建。
