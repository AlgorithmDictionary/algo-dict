# linear program


(定义)



定义:
用下列形式表示的问题。给定一个n × m的实矩阵A, m向量b和n向量c，确定minx{c·x | Ax≥b和x≥0}，其中x在所有n个向量上取值，并且不等式是按分量解释的，即x≥0意味着x的项是非负的。
