# Las Vegas algorithm


(算法技术)



定义:
一种总是产生正确结果的随机算法，每次运行的唯一变化是运行时间。



概括(我是一种……)
随机算法。



专业化(…是一种我。)
跳跃表插入，随机选择枢轴的快速排序变体，随机搜索，Grover算法，Czech, Havas和Majewski的保序最小完美散列，von Neumann的无偏抛硬币算法。



另请参阅
蒙特卡罗算法，ZPP
