# leftist tree


(数据结构)



定义:
用二叉树的一种变体实现的优先级队列。每个节点都有一个计数，它表示到最近的叶节点的距离。除了堆属性之外，左树还被保留，以便每个节点的右后代与叶子的距离更短。



概括(我是一种……)
优先队列。



聚合子(…)是我的一部分或在我身上使用。)
二叉树，堆属性。



注意:
这些被称为“左”，因为左子树通常比右子树高。
