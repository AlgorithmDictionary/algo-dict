# level-order traversal


(算法)



定义:
按深度处理树的所有节点:首先是根，然后是根的子节点，等等。相当于从根开始的宽度优先搜索。



另请参阅
后序遍历，预序遍历，树遍历，cup - giannini树遍历，级别(1)。
