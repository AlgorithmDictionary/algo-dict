# longest common subsequence


(经典问题)



定义:
查找两个或多个字符串的最大长度(或最大权重)子序列的问题。



也称为最重公共子序列。



另请参阅
Ratcliff/Obershelp模式识别，最长公共子串，最短公共超序列。



注意:
最长公共子串是连续的，而最长公共子序列则不必是连续的。
