# load factor


(定义)



定义:
哈希表中元素的数量除以槽的数量。通常写为α (α)



注意:
负载系数越高，检索速度越慢。对于开放寻址，负载因子不能超过1。采用链式时，载荷系数往往超过1。在[CLR90，第224页]之后。
