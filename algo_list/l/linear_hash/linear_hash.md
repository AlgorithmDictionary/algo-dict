# linear hash


(算法)



定义:
一个数字函数，它在改变输入键的间距的同时保持输入键的顺序。



形式定义:S中键的哈希函数f，使得k1, k2∈S∧k1 > k2→f(k1) > f(k2)。



也称为保序散列。



概括(我是一种……)
哈希函数。



专业化(…是一种我。)
保序最小完美哈希。



聚合父级(我是…的一部分或在…中使用)
网格文件，散列堆。
