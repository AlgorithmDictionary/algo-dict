# 编码树


(数据结构)



定义:
表示一种编码的二叉树，如霍夫曼编码产生的二叉树。每一片叶子都是一个编码符号。从根节点到叶节点的路径就是它的码字。



概括(我是一种……)
全二叉树，trie。



聚合父级(我是…的一部分或在…中使用)
霍夫曼编码。
