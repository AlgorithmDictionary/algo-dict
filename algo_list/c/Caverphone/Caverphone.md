# Caverphone


(算法)



定义:
一种对英文名字进行语音编码的算法。



概括(我是一种……)
语音编码算法。



另请参阅
双变音位，Jaro-Winkler, NYSIIS, soundex, Levenshtein distance。



注意:
第一个Caverphone算法类似于变音，但为Caversham数据集(1893-1938年新西兰达尼丁南部地区的名字和口音)定制。更新后的算法2.0是一个“通用英语语音匹配系统”。
