# 并发读，并发写


(定义)



定义:
一种并行内存模型，其中多个处理器可以同时从单个内存位置读取数据，多个处理器可以同时向单个内存位置写入数据。



也被称为CRCW。

并发读、独占写、独占读、并发写、独占读、独占写。
