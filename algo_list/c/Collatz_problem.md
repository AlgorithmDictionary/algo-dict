# Collatz问题


(经典问题)



定义:
考虑函数:如果N是奇数，3 × N + 1;N / 2。从任何正整数开始，反复应用函数总是得到1吗?



注意:
1937年，德国数学家Lothar Collatz首次提出。20世纪50年代开始印刷。

生成的整数序列被称为“冰雹”，因为它们可能上升和下降，但(大概)总是落在地上(整数1)。也称为Syracuse problem和许多其他名称。
