# 元胞自动机


(定义)



定义:
通常是简单有限状态机的二维组织，其下一个状态取决于它们自己的状态和它们的八个最近邻的状态。一般来说，机器可以被安排在更高或更低维度的网格中，具有更大的邻域，或者是任意复杂的处理器。



注意:
康威的《生命游戏》可能是元胞自动机最著名的例子。
