# 聚类


(定义)



定义:
使用开放寻址的哈希表中的条目倾向于存储在一起，即使表有足够的空白空间将它们展开。



另请参阅
主要聚类，次要聚类，k-聚类，无聚类。



注意:
在机器学习中，“聚类”意味着将相似的对象分组在一起，通常是为了确定有意义的类或概念。源自Keith Cassell, 2005年12月29日。
