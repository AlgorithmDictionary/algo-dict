# 组合


(算法)



定义:
从n个元素中选择m，其中m≤n。



另请参阅
排列,子集。



注意:
一个组合是一个恰好有m个元素的子集。

有n!/(n-m)!m!n个(可区分的)元素的组合，每次取m个。为什么?首先，有n个!排列。只有n-m个元素的顺序不同的排列是相同的组合，所以除以(n-m)!取m个元素的顺序也不重要，所以除以m!,太。
