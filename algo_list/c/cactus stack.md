# 仙人掌堆


(数据结构)



定义:
一种仙人掌堆的变体，其中另一仙人掌堆可以附着在顶部。附加的堆栈称为“分支”。当分支变为空时，它将被移除。如果有分支，则不允许Pop。分支只能通过原始引用访问;它不能通过堆栈访问。
