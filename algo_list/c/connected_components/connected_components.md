# 连接组件


(定义)



定义:
无向图的最大连通分量的集合。



概括(我是一种……)
分区。



另请参阅
连通图，双连通分量，无向图，子图，团，强连通分量。



注意:
如果一个图是连通的，那么它只有一个连通的组件。通常使用术语“组件”，并理解“连接”属性。

设G=(V, E)为图，G1=(V1, E1)…，Gm=(Vm, Em)为图的连通分量。每个顶点都恰好在一个连通分量中，即分量划分(1)v。形式上，对于所有i≠j, Vi∩Vj=ø。更进一步，V=V1∪…∪Vm, E=E1∪…∪Em。
