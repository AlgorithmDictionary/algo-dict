# 完备图


(定义)



定义:
在每对顶点之间都有一条边的无向图。



概括(我是一种……)
无向图，密集图，连通图。



专业化(…是一种我。)
小团体。



另请参阅
稀疏图，完全树，完美二叉树。



注意:
一个完全图有n(n-1)/2条边，其中n是顶点的数目。
