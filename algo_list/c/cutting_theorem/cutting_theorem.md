# 切割定理


(定义)



定义:
对于任意集合H (n个超平面)，任意参数r, 1≤r≤n，总存在一个大小为O(rk)的(1/r)切割。在二维空间中，大小为s的(1/r)切割是将平面划分为s个不相交的三角形，其中一些是无界的，使得分区中的三角形与h中的n/r条线相交不超过n/r条。这样的切割可以在0 (nrk-1)时间内计算出来。
