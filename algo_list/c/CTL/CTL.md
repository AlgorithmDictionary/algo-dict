# CTL


(定义)



定义:
一种命题的、分支时间的时间逻辑，其公式可以在线性时间内检验。计算树逻辑的缩写。



注意:
CTL与CTL*的不同之处在于，在CTL路径中量词(E和A)总是与时间运算符(F、G、X、U等)一起出现。因此CTL公式是状态公式。
