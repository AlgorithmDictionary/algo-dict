# 关键路径问题


(经典问题)



定义:
在一个有向无环图中找到从任意源到任意汇聚的最长路径，该图在顶点上有权重或数值。



注意:
这可以用DAG最短路径算法的一种变体来解决，方法是分配一个负无穷(-∞)的初始距离，如果dist(v) + weight(u) > dist(u)，则更新距离和前导，并记录到目前为止找到的最大距离。
