# 集团


(定义)



定义:
无向图中的一组顶点，其中每对顶点之间都有一条边。换句话说，就是一个完备的子图。



概括(我是一种……)
完整的图形。



另请参阅
集团的问题。



注意:
标准读音应该是“cleek”，与antique、boutique、critique、oblique、physic、pique、technique、unique等押韵。

定义源自Lior Malka, 2003年7月31日。
