# 容量设施位置


(经典问题)



定义:
给定一组需求点，一个距离函数和一个参数p，找到一组p个供应对象(点，线，段等)，使某个距离目标函数最小化，并且没有供应支持太多的需求点。



另请参阅
设施的位置。
