# 能力


(定义)



定义:
可以通过边或顶点发送的最大流量。



另请参阅
饱和边缘，流动网络。



注意:
形式上，容量只是一个权重，但由于它与网络和流一起使用，因此将其视为类似于物理容量是有帮助的。
