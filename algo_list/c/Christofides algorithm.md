# Christofides算法


(算法)



定义:
(1)一种寻找旅行商问题近最优解的启发式算法。步骤1:寻找最小生成树t。步骤2:寻找奇数次顶点之间的完美匹配M。步骤3:将M和T的边组合成一个多图g。步骤4:在g中找到一个欧拉环。步骤5:跳过已知的顶点，将欧拉环转化为哈密顿环。(2)求图的色数的算法。



概括(我是一种……)
启发式。



聚合子(…)是我的一部分或在我身上使用。)
最小生成树，完美匹配，多图，欧拉循环。
