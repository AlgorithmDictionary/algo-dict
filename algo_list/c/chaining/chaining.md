# 链接


(数据结构)



定义:
一类冲突解决方案，其中链表处理哈希表中的冲突。两个主要的子类是分离链和合并链，其中列表在表外，列表在表内。



概括(我是一种……)
冲突解决方案。



专业化(…是一种我。)
合并链，分离链。



聚合子(…)是我的一部分或在我身上使用。)
加载因子。



另请参阅
开放寻址。



注意:
可以使用任何可搜索的数据结构来代替链表，但通常最好扩展散列表，从而降低负载因子，并使用简单的冲突解决机制，如链表。
