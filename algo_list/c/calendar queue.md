# calendar queue


(数据结构)



定义:
一个快速的优先级队列实现，有N个桶，每个桶宽度为w，或者覆盖w时间。优先级p比current高的项放入桶(p/w)%N。选择N和w，每个桶中只有很少的项目。保持项目在桶中分类。如果物品数量增加或减少很多，将N加倍或减半，并改变w。



另请参阅
桶排序。
