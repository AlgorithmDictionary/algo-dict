# 完全二叉树


(数据结构)



定义:
一种二叉树，除最深处外，每一层(深度)都被完全填充。在深度n处，也就是树的高度，所有的节点都必须尽可能的向左。



概括(我是一种……)
完全树，二叉树。



专业化(…是一种我。)
二叉堆，完美二叉树。



另请参阅
全二叉树，可扩展哈希，堆。



注意:
一个完全二叉树在每个深度k < n, 2n和2n+1-1之间总共有2k个节点。它可以有效地实现为一个数组，其中索引i处的节点在索引2i和2i+1处有子节点，在索引i/2处有父节点，索引为1。如果子索引大于节点数，则子索引不存在。
