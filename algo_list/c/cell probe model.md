# 细胞探针模型


(定义)



定义:
一种计算模型，其中计算的代价是通过对单元大小为≤log n²位的随机存取存储器的访问总数来衡量的。所有其他计算都不计算在内，被认为是免费的。



另请参阅
随机存取机，指针机，图灵机，大0符号。
