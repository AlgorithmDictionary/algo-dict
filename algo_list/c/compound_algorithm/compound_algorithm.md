# 复合算法


(定义)



定义:
一种算法，其中一个或多个可计算步骤是执行另一个算法的调用。



概括(我是一种……)
算法。



另请参阅
原始的算法。



注意:
任何算法要么是原始算法，要么是复合算法。原语和复合语之间的区别多少有些主观。
