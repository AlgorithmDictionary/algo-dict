# 补
 

(定义)



定义:
(1)布尔值，1为0,0为1。看到没有。(2)集合a，集合中所有的元素都在宇宙中，但不在a中。



另请参阅
差，并，交。



注意:
补可以看作是集合与全集或全称集合之差。即A' = U - A。若全域为{1,2,3,4}，则{1,2}的补数为{3,4}，{3}的补数为{1,2,4}。
