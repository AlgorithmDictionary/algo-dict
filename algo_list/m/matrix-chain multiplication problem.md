# matrix-chain multiplication problem


(经典问题)



定义:
给定一个矩阵序列，使得任何矩阵都可以与前一个矩阵相乘，找出最佳关联，使得用最少的算术运算次数得到结果。人们可以使用动态规划来找到最佳关联。
