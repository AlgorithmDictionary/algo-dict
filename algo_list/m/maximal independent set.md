# maximal independent set


(经典问题)



定义:
图中的一组顶点，对于任意一对顶点，它们之间没有边，并且不能再添加其他顶点，它仍然是一个独立的集合。



注意:
判定图中是否存在大小为k的最大独立集是np完全的。
