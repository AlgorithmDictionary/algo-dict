# minimum spanning tree


(定义)



定义:
加权图中的最小权树，它包含了图的所有顶点。



也被称为MST，最短生成树，SST。



概括(我是一种……)
生成树。



聚合父级(我是…的一部分或在…中使用)
Christofides算法(1)



另请参阅
Kruskal算法，Prim-Jarnik算法，Boruvka算法，Steiner树，树洞，类似的问题:所有对最短路径，旅行推销员。



注意:
最小生成树可用于快速找到旅行商问题的近最优解。

术语“最短生成树”可能在运筹学领域更常见。

斯坦纳树允许额外的连接点，以进一步缩短总长度。
