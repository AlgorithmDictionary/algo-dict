# Munkres' assignment algorithm


(算法)



定义:
通过标记和取消标记条目，覆盖和揭示行和列，在多项式时间内解决分配问题。



也被称为匈牙利算法。


作者:钢


詹姆斯·蒙克斯，1957年。
