# moderately exponential


(定义)



定义:
计算量m(n)(通常是执行时间或内存空间)大于任何多项式nk，但小于c > 1的任何指数cn。形式上，如果∀k > 0 m(n)=Ω(nk)且∀ε > 0 m(n)=o((1+ε)n)，则m(n)是适度指数增长的。
