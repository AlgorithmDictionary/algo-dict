# matrix multiplication


(定义)



定义:
一种取一对矩阵的二进制运算。第一个矩阵的列数必须等于第二个矩阵的行数。



形式定义:如果A是一个n × m矩阵，B是一个m × p矩阵，它们的乘积AB是一个n × p矩阵，其中ABij = Σk=1m (Aik × Bkj)。
