# 盲排序


(算法)



定义:
一种专门的排序算法，它首先构建一个盲树，然后从左到右遍历树。



概括(我是一种……)
排序。



聚合父级(我是…的一部分或在…中使用)
后缀数组。



聚合子(…)是我的一部分或在我身上使用。)
盲目尝试，深度优先搜索。



另请参阅
treesort(1)。



注意:
在专门的算法中用于构建后缀数组。
