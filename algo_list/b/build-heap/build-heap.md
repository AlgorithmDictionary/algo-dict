# build-heap


(算法)



定义:
通过在靠近根的地方逐步执行heapify，将数组转换为堆。对于有n个节点的数组，在比较模型下，这需要O(n)时间。



聚合父级(我是…的一部分或在…中使用)
堆排序,JSort。



聚合子(…)是我的一部分或在我身上使用。)
heapify。
