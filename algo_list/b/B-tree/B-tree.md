# b -树


(数据结构)



定义:
之间的平衡搜索树中,每个节点都有⌈m / 2⌉和m的孩子,其中m > 1是一个固定的整数。M是阶数。根节点可能只有2个子节点。如果树的大部分在慢内存(磁盘)中，这是一个很好的结构，因为通过选择一个大的m，可以使高度和访问次数保持较小，比如1或2次。



也被称为平衡多路树。



概括(我是一种……)
平衡树，搜索树。



专业化(…是一种我。)
2-3-4树，B*树，2-3树。



另请参阅
B+树，多路树，多维索引的ub树，外部存储器数据结构。
