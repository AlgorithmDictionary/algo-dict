# BB(alpha)树


(数据结构)



定义:
一种二叉树，其中每一个子树ρ(T')的平衡以α≤ρ(T')≤1-α为界。



也被称为权重平衡树。



概括(我是一种……)
二叉树，平衡树。



专业化(…是一种我。)
D-tree。



另请参阅
高度平衡树。
