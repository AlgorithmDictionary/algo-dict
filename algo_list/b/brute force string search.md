# 蛮力字符串搜索


(算法)



定义:
通过每次尝试一个位置来在另一个字符串或文本体中查找字符串。有许多更快的字符串匹配算法。



也称为朴素字符串搜索。



概括(我是一种……)
字符串匹配，蛮力。



聚合父级(我是…的一部分或在…中使用)
strsrch。
