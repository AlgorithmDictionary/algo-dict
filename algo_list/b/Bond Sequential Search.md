# Bond顺序搜索


(算法)



定义:
使用键的按位或按位值作为搜索键，一次搜索数组或列表中的两个键。当找到可能的匹配项时，将每个键与项进行比较。



概括(我是一种……)
搜索。



聚合子(…)是我的一部分或在我身上使用。)
线性搜索。



注意:
由于一次遍历可以找到两个键，平摊运行时间大约是线性搜索的一半。
