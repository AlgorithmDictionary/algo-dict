# 暴力字符串搜索不匹配


(算法)



定义:
从字符串中(最左边)的位置开始，依次尝试每个位置，找出模式和从该位置开始的子字符串不匹配的字符数(汉明距离)。返回第一个不匹配数不超过k的位置。



概括(我是一种……)
字符串匹配与不匹配，蛮力字符串搜索。
