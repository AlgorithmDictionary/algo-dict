# bitonic sort


(算法)



定义:
并行地比较和交换元素对(如有必要)。对子集进行排序，然后合并。



也称为批排序。



概括(我是一种……)
的算法。



注意:
这需要O((log n)2/2)个阶段(或步骤)，每个阶段有n/2个比较器。

它对越来越大的混合子集进行排序，有点像Shell排序，并对子集进行合并，就像归并排序。
