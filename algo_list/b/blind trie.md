# 盲试


(数据结构)



定义:
一种特殊的Patricia树，其内部节点仅存储整数k，这是子节点中字符串的公共前缀的长度。同样，两个字符串首先在第(k+1)个字符上不同。



概括(我是一种……)
帕特丽夏树。



聚合父级(我是…的一部分或在…中使用)
盲目的排序。



注意:
用于在工程轻量级后缀数组构建算法中描述的专门算法中，以构建后缀数组。

这个名字来自于检索，读作“树”。请参阅trie的历史注释。
