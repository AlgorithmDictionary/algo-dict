# 遗传算法


(定义)



定义:
一种算法，其行为在设计上独立于对同一问题的典型算法产生影响的某些性质。



概括(我是一种……)
算法。



专业化(…是一种我。)
双调的排序。



注意:
例如，不管输入是什么，遗忘排序算法总是进行相同的比较。这对于外部排序很有用，因为有一个固定的I/O调度，或者并行算法，比如bitonic排序。

其他类型的无关算法是缓存无关算法，它不依赖于硬件参数，如缓存大小和缓存线长度，以及无关路由，它路由数据包独立于网络拓扑、流量历史或拥塞。
